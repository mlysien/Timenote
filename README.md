# Timenote
Simple platform for logging work time. 
This project is my sandbox for TDD approach.

# Feature list
- Log work time
- Edit work time
- Remove work time 
- Permissions 
- Add projects
- Edit projects
- Remove project
- Generate reports
    - weekly
    - monthly
- Approve work time 

# Tech stack
- .NET 9
- NUnit
